Common Bugs

Intro Cart Bug

Carts in Skyrim are a dangerous form of transit, as shown by this glitch that most players have experienced at some point. This can be one of the toughest bugs to iron out for any players using mods, but if you're playing vanilla you should definately try installing one of these mods to try and fix it quickly:

  1. Prisoner cart opening fix SE and AE, a specific fix.
  2. Unofficial Skyrim Special Edition Patch, a mod that aims to fix most of the game's glitches. It may fix the cart if the first one didn't work.
  3. Alternate Life - Live Another Life, for the quitter. This mod will allow you to avoid the scene alltogether if you are not able to fix it and you just want to play.

Infinite loading screen

There is a multitude of reasons for Skyrim to get stuck while loading the game resulting in an infinite loading screen, or "ILS". To help you fix some of the most common causes here's a list of possible solutions:

  1. Change memory allocation. The main reason for the ILS is the game not having enough memory to load the save. To fix this, you can make use of Skyrim Script Extender. If you didn't install it yet you can do it from the SKSE page. With this installed, you can edit the file SafetyLoad.ini in "Skyrim/data/SKSE/Plugins/" and change the value of EnabledOnlyLoading to true.
  2. Reinstall Skyrim.
  3. Udpdate Graphics Drivers.

For more in-depth instructions of these solutions you can see this web.

Crash to desktop

There is a multitude of reasons for Skyrim to crash to desktop, or "CTD". To help you fix some of the most common causes, or at least avoid the CTD as long as possible, here's a list of possible solutions:

  1. Reinstall Skyrim, if the CTD is happening on launch.
  2. Use a manual save, if it happens on load, because sometimes the auto or quick saves can sometimes be faulty.
  3. Set your shadow settings to low, this is one of the more frequent causes for CTDs. To do this, use the game's launcher, go to options and then advanced graphic settings.
  4. Adjust your Windows sound settings. This is an official fix by Bethesda and used to be one of the main problems when booting the game for the first time. To change the setting you can right-click on the volume icon, go to audio settings, choose the audio output device and change the quality to 24bit 44100 Hz.
  5. Udpdate Graphics Drivers.
  6. Install the latest patch of the game, but only if you are not using any mods at all.

For more in-depth instructions of these solutions you can see this web.
