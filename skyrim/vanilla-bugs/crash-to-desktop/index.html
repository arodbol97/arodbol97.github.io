Skyrim Troubleshooting

Crash to Desktop

There is a multitude of reasons for Skyrim to crash to desktop, or "CTD". To help you fix some of the most common causes, or at least avoid the CTD as long as possible, here's a list of possible solutions:

  1. Reinstall Skyrim, if the CTD is happening on launch.
  2. Use a manual save, if it happens on load, because sometimes the auto or quick saves can sometimes be faulty.
  3. Set your shadow settings to low, this is one of the more frequent causes for CTDs. To do this, use the game's launcher, go to options and then advanced graphic settings.
  4. Adjust your Windows sound settings. This is an official fix by Bethesda and used to be one of the main problems when booting the game for the first time. To change the setting you can right-click on the volume icon, go to audio settings, choose the audio output device and change the quality to 24bit 44100 Hz.
  5. Udpdate Graphics Drivers.
  6. Install the latest patch of the game, but only if you are not using any mods at all.
